Features / Specifications

The CMF Phone 1 is a budget-friendly smartphone offering premium-like features such as a 120Hz AMOLED display, powerful 5G processor, and clean software experience.

Feature Details
Display 6.67-inch Super AMOLED, 120Hz refresh rate, Full HD+ resolution.
Processor MediaTek Dimensity 7300 5G (4nm process).
RAM 6GB / 8GB LPDDR4 RAM
Storage 128GB / 256GB UFS 2.2 (expandable up to 2TB)
Camera 50MP rear camera + 2MP depth sensor, 16MP front camera
Battery 5000mAh non-removable
Charging 33W fast charging
OS Nothing OS 2.6 based on Android 14
Connectivity 5G, Wi-Fi 6, Bluetooth 5.3, USB Type-C
Colors Black, Blue, Light Green, Orange

The CMF Phone 1 features a 120Hz Super AMOLED display with high peak brightness for better outdoor visibility.

The MediaTek Dimensity 7300 5G processor provides balanced performance and power efficiency for daily tasks and gaming.

The 50MP rear camera provides good image quality with AI enhancement for better contrast and brightness.
